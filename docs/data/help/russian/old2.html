Более поздние обновления

Раннние обновления 2

2.2.6

В текущей версии произошли следующие изменения:

  • Изменена система нумерации версии программы (убрано последнее число, так как оно не несло смысловой нагрузки).

  • Проверка результатов при загрузке сделана опциональной.

  • Добавлена возможность обновить базу данных о картинках в ручном режиме.

  • Убрана автоматическая проверка базы данных о картинках.

Также исправлены ошибки:

  • Утечка памяти при поиске.

  • Некорректное прерывание процесса загрузки базы данных с картинками.

  • Неверное состояние на панели статуса во время поиска.

2.2.5.531

В текущей версии исправлена ошибка:

  • Программа выдавала не нулевую разность для абсолютно одинаковых ихображений.

2.2.4.528

В текущей версии произошли следующие изменения:

  • Добавлен перевод на белорусский язык.

  • Добавлен перевод на традиционный китайский язык.

Также исправлены ошибки:

  • Некоторые картинки в формате JPEG ошибочно определялись как дефектные.

  • Не вызывалась справка, если путь к программе содержал пробелы.

  • Картинки некорректно отрисовывались после изменения размера окна программы.

  • Программа не отображала картинки, у которых есть пометка "только для чтения".

2.2.3.520

В текущей версии исправлена критическая ошибка:

  • Картинки не удаляютсяи не переименовываются.

2.2.2.517

В текущей версии произошли следующие изменения:

  • Ускорен алгоритм загрузки изображений.

  • Добавлена возможность отключения горячих клавиш.

  • Теперь у приложения уменьшается приоритет во время работы в фоновом режиме.

  • Увеличено максимально возможное число потоков сравнения и загрузки.

  • Добавлена возможность вызова справки из вспомогательных окон.

Также исправлены ошибки:

  • Моргание картинок при смене текущего результата.

  • Иногда память не освобождалась после окончания поиска.

  • Ошибка при отображении некоторых файлов с GIF-анимацией.

2.2.1.503

В текущей версии произошли следующие изменения:

  • Теперь поиск и другие длительные процессы можно прервать нажатием клавиши Escape.

  • Вместо одной базы данных о картинках, теперь используется отдельные базы данных для каждого значения параметра "Нормированный размер картинок", что позволяет экономить память.

  • Убран режим работы программы, когда база данных постоянно находилась в оперативной памяти.

  • Параметр "Нормированный размер картинок" теперь может принимать значение 128.

Также исправлены ошибки:

  • Неправильная сортировка результатов.

  • Некорректная загрузка поврежденных и нестандартных файлов в формате Jpeg2000.

  • В Windows 7 не выделялась текущая картинка при открытии родительского каталога.

  • Программа не отображала картинки с длинными именами.

  • Поиск прерывался, если пользователь нажимал клавишу пробел.

2.2.0.495

В текущей версии произошли следующие изменения:

  • Добавлена поддержка изображений в форматах JP2(Jpeg2000) и PSD.

  • В колонке "Разрешение" результаты теперь сортируются по площади изображения, а не по ее ширине.

  • Теперь после поиска программа сортирует результаты таким же образом, как они были сортированы до этого.

  • Опционально программа может не сравнивать друг с другом картинки внутри одного каталога.

  • При сравнении изображений, программа может опционально игнорировать край картинок.

  • В окно редактирования путей добавлены горячие клавиши (Control-C, Control-V, Control-X и Delete).

Также исправлены ошибки:

  • Программа аварийно завершала свою работу, если пользователь сначала удалил последний результат в таблице, а затем попытался начать выделение результатов при помощи клавиши Shift.

  • Программа в некоторых случаях неправильно группировала результаты.

  • Грамматические ошибки в русской версии справки.

2.1.1.466

В текущей версии произошли следующие изменения:

  • Добавлен механизм Drag&Drop для добавления файлов и директорий в окне настройки путей.

  • Теперь окно прогресса не появляется в случае коротких (продолжительностью менее 200 мс) операций.

  • При переименовании картинок добавляется расширение файлов, если пользователь сам забыл это сделать.

  • Добавлен перевод на упрощенный китайский язык.

Также исправлены ошибки:

  • Программа ошибочно использовала данные о картинках, собранные предыдущей версией программы.

  • Пользователь мог ввести некорректные значения параметров в окне опций.

  • Файлы результатов и база данных об ошибках некорректно сохранялись, если пользователь переименовывал или перемещал изображения в программе.

  • Переименовывание или перемещение файлов приводило к потере данных об этих файлах в базе данных об ошибках.

2.1.0.452

В текущей версии произошли следующие изменения:

  • Изменен алгоритм сравнения изображений (для увеличения точности программы).

  • Появилась возможность опционально ограничить количество результатов поиска (для экономии ресурсов).

  • Теперь можно регулировать количество отменяемых пользователем операций (для экономии ресурсов и для ускорения работы программы).

  • Опция 'Пороговая разность' вынесена на главную панель инструментов.

  • Изменен алгоритм удаления временных файлов.

  • Ограничена возможность пользователю останавливать некоторые операции (для сохранения целостности данных).

  • Добавлено окно прогресса для операций с текущим результатом.

  • В отображаемую под картинками информацию добавлен тип изображений.

  • Если у пары картинок различается тип, то он теперь выделяется красным цветом.

  • Переименовать изображение теперь можно при помощи двойного щелчка мышью по имени картинки на панели просмотра.

  • Добавлен перевод на немецкий язык.

2.0.0.431

В текущей версии произошли следующие изменения:

  • Добавлена поддержка 64 битности (теперь программа может сканировать коллекции картинок, содержащие более 1 миллиона изображений).

  • Появилась возможность отмены/повтора последних 10 действий пользователя.

  • Теперь в программе можно переименовывать и перемещать изображения.

  • Теперь пары изображений могут опционально отображаться с учетом их относительного размера.

  • Малые изображения теперь опционально не растягиваются до размеров панели просмотра.

  • Теперь разность для полностью идентичных изображений выделяется зеленым цветом.

  • Убрана возможность одновременного замещения для нескольких пар изображений (т.к. это могло привести к неопределенным результатам).

Также обновлена справка и исправлены ошибки:

  • Программа некорректно отображает процент выполненной работы.

  • Программа на Windows 7 не корректно отбражает результаты, если ее свернуть во время поиска.

  • Программа иногда падает при поиске.
