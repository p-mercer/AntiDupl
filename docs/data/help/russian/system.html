Системные требования

Требования к операционной системе:

  • Windows Vista, Windows 7, Windows 8.

  • Windows 2000 Service Pack 3, Windows Server 2003 и Windows XP Service Pack 2 при установке на них Microsoft.NET Framework 2.0 (22 Мб) или более поздней версии.

  • Для операционной системы Windows 2000, возможно потребуется так же добавить в системный каталог файл gdiplus.dll. Найти его можно, например, здесь.

Требования к свободному месту на диске:

  • При установке Microsoft.NET Framework 2.0 может потребоваться 280 Мб (x86) или 610 Мб (x64).

  • Кроме того, самой программе требуется место для сохранения данных о ранее сканированных изображениях. Это приблизительно 2 Кб на одно изображение (при типичных настройках поиска).

Требования к оперативной памяти:

  • Самой программе требуется приблизительно 50 Мб.

  • Кроме того, программе требуется память для хранения данных о ранее сканированных изображениях (приблизительно 2 Кб на одно изображение).

Требования к центральному процессору:

  • Желательно, чтобы ваш процессор мог выполнять инструкции SSE2 (их поддерживают все процессоры Intel начиная с P-IV, и все процессоры AMD начиная с Athlon 64).

  • Наличие нескольких ядер/процессоров у вашего компьютера может значительно ускорить процесс работы программы.

  • Высокая частота только приветствуется.
