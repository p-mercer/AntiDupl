Последние обновления

2.3.9

  • Программа переехала на GitHub.

2.3.8

  • Поменяны местами изображения соседних картинок в горизонтальном виде.

  • Исправлена загрузка подписи горячей клавиши в меню при использовании профиля созданного в старой версии.

2.3.7

  • Добавлена подсветка различий изображений.

  • Добавлена опция "Показывать соседние изображения".

  • Добавлены опции: "Загружать профиль при загрузке", "Сохранять профиль при закрытие".

  • Исправлено не работающее удаление обоих изображений.

2.3.6

  • Добавлена опция "Сравнивать картинки внутри одного каталога".

  • Пункт меню "Обновить базу данных о картинках" поменялся на "Удалить неактуальные записи из базы данных о картинках". Теперь он проверяет существуют ли файлы для которых хранятся эскизы в базе данных на прежнем месте. Если нет, то неактуальная запись удаляются. Позволяет уменьшить размер базы данных после операций удаления или перемещения.

  • Добавлена колонка отображения размера группы (количество картинок в группе).

  • В контекстное меню для картинки добавлены пункты занесения её или её директории в список игнорируемых (пропускаемых) путей.

2.3.5

  • Ускорен подсчет SSIM с помощью SSE2.

2.3.4

  • В контекстное меню для выделенных файлов добавлены пункты: "Заместить", "Переименовать как" и "Переместить к".

  • Добавлена в контекстное меню изображений входящие в группы, содержащих больше двух файлов, возможность группового переименования и перемещения файлов группы.

  • Устранены некоторые утечки памяти при считывание EXIF информации.

  • Изменен алгоритм подсчета размытия.

2.3.3

  • Добавлен новый алгоритм сравнения изображений с помощью индекса структурного сходства - SSIM. Он позволяет найти дубликаты среди сильно измененных картинок, а также лучше сравниваются изображения с большим однородным фоном.

  • Добавлено считывание и отображение во всплывающей подсказке EXIF информации по основным полям.

  • Исправлено падение при обработке больших файлов и недостатке памяти, теперь при недостатке памяти эти файлы пропускаются.

2.3.2

В текущей версии произошли следующие изменения:

  • Добавлен подсчет блочности и размытости изображения.

  • Добавлены пункты меню "Переименовать как соседнею" и "Переместить и переименовать к соседней".

  • Добавлена возможность указания поиска в поддиректориях для каждой директории в окне выбора путей.

2.3.1

В текущей версии произошли следующие изменения:

  • Добавлен перевод на белорусский язык.

  • Добавлен перевод на немецкий язык.

  • Пункты меню "Открыть профиль поиска" и "Сохранить профиль поиска как" продублированы на панели инструментов.

Также исправлены ошибки:

  • Пользователь не мог сохранить профиль поиска при нулевом количестве результатов.

  • Пороговая разность на панели инструментов не обновлялась при загрузке профиля поиска.

  • Иногда настройки программы не сохраняются.

  • Программа не может загрузить результаты содержащие картинки с дефектами.

2.3.0

В текущей версии произошли следующие изменения:

  • Добавлена поддержка изображений в форматах DDS(DirectDraw Surface) и TGA(Truevision Graphics Adapter).

  • Добавлена поддержка профилей поиска.

  • Добавлено опциональное ограничение на максимальный размер картинок при поиске.

  • Добавлено сохранение/востановление позиции главного окна программы.

  • Теперь при прочих равных условиях программа будет рекомендовать удалить более новые картинки.

Также исправлены ошибки:

  • Программа открывала некорректную директорию при переименовании картинок.

  • Программа при переименовании не перемещала картинки на другой диск.

2.2.11

В текущей версии произошли следующие изменения:

  • Добавлен перевод на украинский язык.

  • Добавлен перевод на немецкий язык.

Также исправлены ошибки:

  • Директория "user" создавалась в директории программы даже если пользователь указал другой путь для сохранения файлов пользователя.

2.2.10

В текущей версии произошли следующие изменения:

  • Добавлена опциональная проверка базы данных об ошибках при загрузке.

  • Добавлено отображение процесса загрузки и сохранения базы данных об ошибках.

  • Сайт программы сменил свой адрес.

2.2.9

В текущей версии произошли следующие изменения:

  • Добавлена частичная загрузка базы данных о картинках.

  • Добавлена поддержка разных версий базы данных о картинках, базы данных об шибках и файла результатов.

  • По умолчанию пользовательские файлы сохраняются в отдельную директорию.

  • Добавлена минимальное время показа стартового окна (3 секунды).

  • Добавлен перевод на упрощенный китайский язык.

2.2.8
