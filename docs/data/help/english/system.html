Requirements

Operating system requirements:

  • Windows Vista, Windows 7, Windows 8.

  • Windows 2000 Service Pack 3, Windows Server 2003, and Windows XP Service Pack 2 with the installation of Microsoft.NET Framework 2.0 (22 Mb) or higher.

  • For Windows 2000 may need to add to the system directory file gdiplus.dll. You can find it, for example, here.

Requirements for free space on disk:

  • When you install the Microsoft.NET Framework 2.0 may require 280 Mb (x86) or 610 Mb (x64).

  • In addition, the program itself needed a place to store data about previously scanned images. This is approximately 2 kb per image (with the typical search settings).

RAM requirements:

  • Program itself requires approximately 50 MB.

  • In addition, the program requires a memory for storing data about previously scanned images (approximately 2 KB per image).

Requirements CPU:

  • It is desirable that your processor can execute instructions SSE2 (to support all processors from Intel, starting with the P-IV, and all processors from AMD Athlon 64).

  • The presence of multiple cores/processors in your computer can significantly accelerate the program work.

  • The high frequency is to be welcomed.
