1.0.0.0

This is the first public version of the program AntiDupl.NET. It actually consists of two parts: algorithmic kernel AntiDupl and graphical shell to him AntiDupl.NET. In the algorithmic core AntiDupl implements search and comparison of images, detection of defects, as well as mechanisms for processing the results. Here are its key features:

  • The advanced parameters of the search:

    • You can search in several directories, so it is possible to compare individual files.

    • You can exclude directories/files from search, as well as can control search of a system and hidden files and folders.

    • Also you can specify the already checked directories/images, which already should not be compared with each other (this can significantly reduce the search time).

    • You can specify a directory/files in which/that is preferable to delete (it effects in the recommendations).

  • There are rich possibilities to handling of search results:

    • You can navigate through the results in random order,

    • Sort by various parameters,

    • Save and load on the disc,

    • Get recommendations for their treatment.

    • The image dupl pairs are grouped into (trios, quartets, etc.).

    • Also you can perform operations on multiple images simultaneously.

  • It is possible to find the mirror and rotated duplicates of images.

  • Searched pictures are recorded in a special base, which greatly accelerates the time of re-search.

  • Comparison of images is performed by multiple threads. Also there is support for SSE2 instructions of the processor. This allows you to use the full capabilities of modern processors.

  • The algorithmic part of the program was implemented as a separate module (dynamic link library). His C-interface has ANSI and UNICODE versions of functions. Thus third party programs can easy use the kernel algorithms.

The shell of the AntiDupl.NET is based on the .NET technology. It provides a user-friendly graphical interface to the algorithmic kernel: you can fast and easy customize your search, view and handle the results.
