Context Menu

The context menu is invoked by right-clicking the mouse on the Table of Results.

  • or - Delete first pictures in selected results. It allows removing all first pictures from duplicative pairs selected by user.

  • or - Delete second pictures in selected results. It allows removing all second pictures from duplicative pairs selected by user.

  • or - Replace firsts pictures by the seconds pictures in selected results.

  • or - Replace seconds pictures by the firsts pictures in selected results.

  • or - Delete all duplicated pictures in selected results. It allows removing all duplicated pictures selected by user.

  • or - Delete defect pictures in selected results. It deletes all defect pictures, who selected by user.

  • or - Rename first pictures as the second pictures in selected results. To rename a name at the first pictures to name the second pictures, if files with such name already are, on the similar.

  • or - Rename second pictures as the first pictures in selected results.To rename a name at the second pictures to name the first pictures, if files with such name already are, on the similar.

  • or - Move first pictures to the second pictures in selected results. To move the first pictures to the directory of the second pictures. If in the directory already there is a file with a name moved, the moved file is renamed.

  • or - Move second pictures to the first pictures in selected result. To move the second pictures to the directory of the first pictures. If in the directory already there is a file with a name moved, the moved file is renamed.

  • - Process selected results automatically. When you click this button, the computer will perform its recommended actions on the group of the results selected by user.

  • - Mark selected results as a mistakes. This button allows saving this selected user group results (defective pictures and duplicative pairs) in database error.

Note: the context menu functionality is dublicated by the Tool Bar. In addition, it should be remembered that the operations invoked from this menu can be undone.
